Shulkers is a Bun + TypeScript CLI for managing Minecraft server plugins/mods. It uses gunshi for CLI command definitions, ky for HTTP requests, jszip/yaml for JAR inspection, and stores project state in .shulkers/project.yml. Main code lives under src/commands for CLI commands and src/lib for project/config/repository/download/version logic. Built-in repositories include Modrinth and Spiget, with optional generic/GitHub repositories.
